TROUBLESHOOTING

卡住時，先從這裡找答案

先讀錯誤訊息，再檢查資料、執行順序與 Runtime 狀態。

為什麼 Colab 找不到資料？

先重新執行第一格。若仍失敗，下載當日 CSV 後上傳到 Colab 的 /content，再由上到下執行。

我可以直接修改所有程式碼嗎？

先完整執行原始版本，只修改有 TODO 或講義指定的欄位。每次只改一項，才知道結果為什麼改變。

勾選的進度會傳給講師嗎？

不會。進度使用 localStorage，只保存在目前瀏覽器。2026 課程已封存，正式驗收與課後批改已結束。

我可以把教材拿去授課或重新上架嗎？

可用於個人學習並分享本網站原始連結；完整重製、重新上架、收費授課或機構內訓，請先取得書面同意。詳見使用說明。

這些數字可以代表企業真實狀況嗎？

不可以。全課程使用虛擬資料，只用來練習分析流程與決策表達。

手機可以上課嗎？

網站可用手機閱讀，但 Colab 編程與圖表操作仍建議使用筆電或桌機。

Runtime 重啟後結果不見了怎麼辦？

重新執行全部儲存格；需要保留的成果請下載，或把 Notebook 副本存到自己的 Google Drive。
